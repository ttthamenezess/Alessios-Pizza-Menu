# 🍕 Alessio's Pizza Menu

Página simples de menu de pizzaria desenvolvida com **HTML5** e **CSS3**.

O projeto foi criado como exercício prático em um curso de Full Stack, com foco em estruturação de página, estilização e organização visual de um cardápio online.

---

🔗 **Deploy:** https://ttthamenezess.github.io/Alessios-Pizza-Menu/

🔗 **Repositório:** https://github.com/ttthamenezess/Alessios-Pizza-Menu

---

## 📌 Visão Geral

O **Alessio's Pizza Menu** é uma página web estática que apresenta um cardápio de pizzaria de forma simples e visual.

A proposta do projeto é praticar a criação de layouts com HTML e CSS, simulando uma página de menu para um estabelecimento gastronômico.

---

## ⚙️ Funcionalidades

- 🍕 Exibição de menu de pizzaria
- 🎨 Estilização personalizada com CSS
- 🖼️ Uso de imagens no layout
- 📄 Página estática simples
- 🌐 Publicação via GitHub Pages

## ▶️ Como Executar o Projeto Localmente

### 1. Clone o repositório

```bash

```

### 2. Acesse a pasta do projeto

```bash
cd Alessios-Pizza-Menu
```

### 3. Abra o arquivo principal

Abra o arquivo abaixo no navegador:

```text
index.html
```

Também é possível executar pelo VS Code usando a extensão **Live Server**.
